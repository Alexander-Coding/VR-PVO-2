  ЧТО СДЕЛАТЬ В UNITY
  1. Убедитесь, что вы в Edit Mode (не в Play Mode).
  2. Меню File → Save Project (или Ctrl+S по сцене).
  3. Если сцена показывала старую версию — закройте её и откройте снова из Assets/Scenes/SampleScene.unity (файл на диске уже с правками).
  4. Запустите игру и проверьте: захват, непрерывная стрельба, перегрев, дым, звук пара.
